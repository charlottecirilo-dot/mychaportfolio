üÜò Common Issues

Images not showing?

  • Check URL is correct
  • Verify image is publicly accessible
  • Use HTTPS URLs, not HTTP

Form not working?

  • Currently just logs to console (check DevTools)
  • Integrate EmailJS or Formspree to make it send emails

Not responsive on mobile?

  • Clear browser cache and refresh
  • Make sure DevTools viewport is set correctly
  • Test on actual mobile device
